            curl
curl -A "Chrome" https://www.baidu.com
curl -X POST https://www.baidu.com
-I 只返回请求头信息
-L跟随重定向请求
-k允许发起不安全的ssl请求
-s 只显示结果
-v 显示连接过程信息
curl -b a=123 http://httpbin.org/cookies     cookies
curl -d test=123 http://httpbin.org/post   -d 以POST方法请求url,并发送相应的的的参数
    -d a=1 -d b=2  | -d a=1&b=2 | -d @filename
curl -O http://http://httpbin.org/image/jpeg  下载文件
curl -o xxxx.jpg "http://httpbin.org/image/jpeg" 自定义命名
curl -o xxxx.jpg  -H 'accept:image/png' "http://httpbin.org/image/jpeg" 自定义接收文件格式


            wget
wget -O xxx.jpg "http://httpbin.org/image/jpeg" 保存
wget --limit-rate=200k xxx.jpg "http://httpbin.org/image/jpeg" 限速
-c 断点续传
-b 后台下载
-U "windows IE 10.0" 设置user-agent
--mirror 镜像网站
-p 下载所有资源
-r 递归

            代理服务器
sudo apt install tinyproxy

    vim /etc/tinyproxy.conf

        ```bash
        # Allow 127.0.0.1
        ```
        然后，修改一下默认端口号
        ```bash
        Port XXXX   (自定义）
        ```
    systemctl restart tinyproxy
